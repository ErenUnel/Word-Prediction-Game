## EN | About the Project
This is my 1st-year term project developed for the **"Introduction to Information Systems"** course at Istanbul Technical University (ITU). The project aims to provide an interactive and dynamic user experience by utilizing core web technologies.

### Features:
* Developed using HTML, CSS, and JavaScript .
* Includes word prediction mechanisms and dynamic content management.

##How to Play
###  Gameplay and Rules:
1. **Entry:** When the page loads, an "About Me" section welcomes the user.
2. **Start:** Click the "Word Prediction" button to switch to the game screen.
3. **Objective:** The goal is to guess a hidden word consisting of 5 letters.
4. **Life System:** You have a total of **3 lives**. Each incorrect letter guess results in the loss of 1 life.
5. **Scoring:** - Each correct letter guess awards **20 points**.
   - You win the game once you reach a total of **100 points**.
6. **Instant Win:** If you correctly guess the entire word directly, you win the game instantly.
